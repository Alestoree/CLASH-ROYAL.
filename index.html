🔑 Cómo obtener tu API Key
1
🌐 Entra al portal de desarrolladores
Abre este link en tu navegador y crea una cuenta gratis:
🔗 developer.clashroyale.com
2
🔎 Conoce tu IP actual
Necesitas saber tu IP para registrarla. Ábrela aquí:
🔗 whatismyip.com
Copia el número que aparece arriba, algo como 187.123.45.67
⚠️ Importante: Si cambias de WiFi, tu IP cambia y deberás actualizarla en el portal. Puedes hacerlo cuando quieras en "My Keys".
3
➕ Crea tu Key
En el portal ve a My Keys → Create New Key.

• Key Name: Cualquier nombre (ej: MiApp)
• Allowed IP Addresses: La IP que copiaste

Haz clic en Create.
4
📋 Copia y pega tu Key aquí
El portal te mostrará un texto largo. Cópialo todo y pégalo en el campo "Tu API Key" de arriba. Luego haz clic en Guardar Key.
🎉
¡Ya puedes buscar jugadores!
Una vez guardada tu key, busca a cualquier jugador del mundo por nombre o por tag (#XXXXXXXX).
